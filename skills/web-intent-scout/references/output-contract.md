# Output Contract

Use this before final delivery.

## Compact Answer

For small searches:

```markdown
Answer:

Sources checked:
- ...

Why this is reliable:

Caveat:

Next step:
```

If uncertainty matters, add:

```markdown
Uncertainty:
```

## Web Scout Dossier

For substantial searches:

```markdown
# Web Scout Dossier

## 1. Search Depth

## 2. Search Assumptions / Orientation Result

## 3. Intent Map

## 4. Research Brief

## 5. Task Type Routing

## 6. Query Families

## 7. Source Notes / Source Ledger

## 8. Source Credibility

## 9. Claim Ledger

## 10. Conflict Resolution

## 11. Freshness Check

## 12. Risk / Bias Check

## 13. Scorecard

## 14. Recommendation

## 15. Remaining Uncertainty
```

## Recommendation Shape

Give practical answers:

- what to use or trust;
- what not to use or trust;
- why;
- what evidence supports it;
- what caveat matters;
- what the user should verify next.

For decisions, structure the conclusion as:

```markdown
Best answer:
Why:
Do not choose / exclude:
Key evidence:
Remaining uncertainty:
Next action:
```

## Citation Rule

Always include links to used sources.

For recommendations, include at least:

- official or primary source when available;
- independent or user-feedback source when relevant;
- risk/freshness source when the decision depends on it.

## Avoid

- link dumps without recommendation;
- overtrusting SEO pages;
- hiding uncertainty;
- presenting old information as current;
- giving medical/legal/financial certainty from weak sources.
- recommending from raw search snippets without a source-note and claim-ledger pass.
