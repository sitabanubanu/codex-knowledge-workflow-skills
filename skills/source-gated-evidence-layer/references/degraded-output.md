# Degraded Output

When source gate blocks full analysis, write a degraded output that answers:

1. What was acquired?
2. What is missing?
3. Why is complete analysis not allowed?
4. What can the user provide next?

Suggested file:

```text
10_video/00_source/degraded_source_report.md
```

The title and opening section must say degraded, blocked, failed, metadata-only,
or secondary-only. Do not make degraded output look like a normal report.
