# 2026-07-15 Knowledge Workflow 工程测评试验

本目录保存 v0.6.0（commit `5fddb2e`）的一轮历史工程试验，包含任务清单、执行脚本、结构化结果摘要和完整报告。

试验覆盖三类任务：

- 材料充分；
- 材料部分充分或只能作为二手说明；
- 获取失败或分析目标与材料范围不匹配。

主要文件：

- `manifest.json`：最低工作流试验的冻结任务清单；
- `search_manifest.json`：搜索试验清单；
- `RESULTS.md`：结构化结果摘要；
- `FULL_EVALUATION_REPORT.md`：当时生成的完整评测报告；
- `PILOT_LIMITATIONS.md`：复核后确认的方法学限制。

这些结果只能视为历史工程试验，不能作为正式 A/B 结论、产品优越性证明或发布级可靠性声明。新的门控评测从 `eval/v2/` 开始，必须使用中性输入、隔离金标和独立结构化计分。
