# Weighting Rules

Use this when deciding what belongs in the main report, appendix, or correction log.

## Weight Classes

| Class | Meaning | Placement |
|---|---|---|
| `Core` | recurring problem or operating rule across domains and time | main report |
| `Current Dominant` | strongest recent 6-12 month line | early main report |
| `Historical Root` | older formative line still influencing current patterns | main report but after current line |
| `Turning Point` | dated change that altered interpretation or action | main report if high impact |
| `Active Thread` | recurring but not central line | secondary section |
| `Supporting Pattern` | reinforces another line | evidence within section |
| `Episode` | bounded event, useful but not defining | appendix or short note |
| `Noise` | unreliable, incidental, or misleading | omit unless preventing an error |

## Evidence And Time

Increase weight when:

- repeated across months or years;
- appears in several domains;
- has direct quotes and behavior evidence;
- user confirms it;
- it changes future action.

Decrease weight when:

- based on one vivid quote;
- old and not supported recently;
- explained by context;
- user provides a clear correction;
- it mainly reflects another person's content or a forwarded chat;
- it does not change report conclusions.

## Placement Rule

Weight is not just a label.

It must affect:

- section order;
- length;
- wording strength;
- whether the topic is main text, appendix, or omitted.

Do not give a full main-report section to an `Episode` just because it is interesting.

## Current Vs Historical

If an old line is vivid but recent evidence points elsewhere:

- keep the old line as `Historical Root` or `Episode`;
- lead with the recent `Current Dominant` line;
- explain the time shift.

## User Corrections

User correction can:

- lower a claim's class;
- split a claim into narrower subclaims;
- turn a model inference into user-confirmed fact;
- mark a topic as not central even if it appears often.

Do not let user correction erase strong contrary evidence without recording the tension.
